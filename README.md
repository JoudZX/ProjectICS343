# NetAssist Enhanced

A complete Flask networking assistant with:

- Subnet Calculator
- DNS Resolution
- Reverse DNS Lookup
- Port Checker
- Domain Validation
- Ping Test
- Domain Path Visualization

## Run

```bash
pip install -r requirements.txt
python app.py
```

Open: http://127.0.0.1:5000
